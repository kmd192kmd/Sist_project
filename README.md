# 🏫 교육센터 운영 관리 시스템 (Oracle Database Project)

쌍용 교육 센터의 과정을 관리하고 교육생의 성적, 출결 및 사후 처리를 통합 관리하는 데이터베이스 기반 시스템입니다.

---

## 📌 1. 프로젝트 개요
- **개발 환경**: Oracle Database 21c / 19c
- **목적**: 교육 과정의 설계부터 모집, 운영(성적/출결), 취업 지원까지의 전체 프로세스 자동화
- **사용자 권한**: 관리자(Admin), 교사(Teacher), 교육생(Student)

---

## 🏛 2. 기초 정보 및 인프라
| 항목 | 상세 내용 |
|:--- |:--- |
| **강의실** | 총 6개 (1~3강의실: 30명 / 4~6강의실: 26명) |
| **교육 과정** | 5.5개월 / 6개월 / 7개월 과정 (연평균 12개 과정 운영) |
| **과목 구성** | 공통 과목(Java, Oracle 등) + 특화 과목(Python, 빅데이터 등) |
| **운영 프로세스** | 과정 정의 → 시행 확정 → 모집/선발 → 개강 → 운영(성적/출결) → 종강 → 사후 처리 |

---

## 🔑 3. 주요 기능 요구사항

### 🛡 관리자 (Admin)
- **기초 정보 관리**: 과정명, 과목명, 강의실(정원), 교재명(출판사) CRUD
- **계정 관리**: 교사 및 교육생 정보 등록/관리 (주민번호 뒷자리 PW 설정)
- **개설 관리**: 과정 및 과목 개설, 강의실 및 교사 배정
- **성적/출결**: 전체 교육생 성적 조회 및 기간별/과정별 출결 현황 모니터링
- **특화 기능**: 교육생 면접 및 선발, 상담일지 관리, 사후처리(취업 확인) 지원

### 👨‍🏫 교사 (Teacher)
- **스케줄 조회**: 강의 예정/중/종료 상태별 스케줄 확인
- **배점 관리**: 과목별 출결(min 20점), 필기, 실기 배점 등록 (합계 100점)
- **성적 관리**: 교육생별 시험 점수 입력 및 성적 등록 여부 관리
- **출결 관리**: 담당 과정 교육생의 근태(정상, 지각, 조퇴 등) 조회 및 관리

### 👨‍🎓 교육생 (Student)
- **정보 조회**: 본인의 수강 과정, 과목 기간, 교재, 강사 정보 확인
- **성적 조회**: 과목별 배점 정보 및 본인의 취득 점수(필기/실기/출결) 조회
- **출결 관리**: 일일 출결(출근/퇴근) 기록 및 본인 출결 현황 조회
- **사후 처리**: 본인의 취업 활동 및 취업 유무 기록

---

## 🗃 4. 데이터베이스 설계 규칙 (Naming Rule)
- **표기법**: `Camel Case` 사용 (예: `tblStudentScore`)
- **접두어 (Prefix)**:
  - 테이블: `tbl_`
  - 뷰: `vw_`
  - 프로시저: `proc_`
  - 함수: `fn_`
  - 트리거: `trg_`
  - 인덱스: `idx_`

---

## 🛠 5. 기술적 구현 계획 (SQL Objects)

### 1️⃣ Tables (DDL)
- `tblMember`, `tblCourse`, `tblSubject`, `tblScore`, `tblAttendance` 등 관계형 모델링 기반 구축

### 2️⃣ Views (DML)
- **교사 강의 스케줄 뷰**: 강의 진행 상태(날짜 기준) 자동 계산 포함
- **교육생 성적 상세 뷰**: 과목별 배점과 점수를 한눈에 확인

### 3️⃣ Stored Procedures
- **성적 입력 프로시저**: 배점 합계(100점) 및 출결 최소 점수(20점) 유효성 검사 로직 포함
- **출결 등록 프로시저**: 출근/퇴근 시간을 기반으로 근태 상태(정상/지각 등) 자동 분류

### 4️⃣ Triggers
- 과정 종료 시 해당 과정 교육생들의 상태를 자동으로 '수료'로 업데이트 (중도탈락자 제외)

## 📈 6. 추가 특화 업무
- **데이터 통계**: 일/주/월/년 단위의 교육생 수료율 및 취업 성공률 통계 기능
- **교사 평가**: 종강 후 교육생들이 실시하는 교사 강의 평가 시스템
- **상담 관리**: 교사와 교육생 간의 정기 상담일지 이력 관리
